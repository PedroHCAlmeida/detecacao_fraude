Limpeza de dados - Contas bancárias fraudulentas

Esse trabalho tem como objetivo realizar uma limpeza na base de dados escolhida. Nesse sentido os seguintes passos serão executados:

  • Renomear colunas
  • Avaliar linhas duplicadas
  • Verificar se existem valores faltantes

Renomear colunas

Uma vez que, as colunas são todas em inglês, para facilitar a leitura do trabalho vou renomeá-las para portguês:

  • Colunas originais

rename_columns = 
  list(
    "fraud_bool" = "fraude",
    "income" = "renda",
    "name_email_similarity" = "email_nome_similaridade",
    "prev_address_months_count" = "meses_endereco_anterior",
    "current_address_months_count" = "meses_endereco_atual",
    "customer_age" = "faixa_idade",
    "days_since_request" = "dias_desde_pedido",
    "intended_balcon_amount" = "transferencia_inicial",
    "payment_type" = "tipo_pagamento",
    "zip_count_4w" = "cep_aplicacao_4w",
    "velocity_6h" = "media_aplicacao_6h",
    "velocity_24h" = "media_aplicacao_24h",

    "credit_risk_score" = "score_credito",
    "email_is_free" = "email_gratis",
    "housing_status" = "status_casa",

    "phone_mobile_valid" = "celular_valido",
    "bank_months_count" = "tempo_conta_anterior",

    "source" = "fonte_pedido",

    "device_os" = "sitema_operacional",
    "keep_alive_session" = "permaneceu_sessao",
    "device_distinct_emails_8w" = "emails_distintos_dispositivo_8w",
    "device_fraud_count" = "aplicativos_fraudulentos",
    "month" = "mes"
  )

fraudes_clean = 
  fraudes |>
  rename_all(
    ~rename_columns[.] |> unlist()
  ) |>
  mutate_at(
    vars(
      fraude, 
      tipo_pagamento, 
      situacao_profissional, 
      email_gratis, 
      status_casa,
      telefone_valido,
      celular_valido,
      possui_outros_cartoes,
      pedido_estrangeiro,
      fonte_pedido,
      sitema_operacional,
      permaneceu_sessao,
      aplicativos_fraudulentos, 
      faixa_idade,
      renda
      ),
    as.factor
  )
